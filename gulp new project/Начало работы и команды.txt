Начало работы
1. Запустить установку сборщика командой npm i
2. В проекте уже создана страница index.html, а также блоки head, header, footer и example.
Они подключены к основной странице для примера
3. Стили страницы находятся в папке scss. К странице можно подключать разные стили, конкретно для данной страницы. Задаются через настройку head
4. Шрифты подключаются автоматически. Для подключения новых нужно удалить fonts.scss из src/scss/base. Там же вносить изменения
5. При создании блока обязательно войти в файл js и очистить его

Основные команды
1. npm run dev - режим разработчика
2. npm run build - собрать проект для передачи заказчику
3. npm run create {name} - создать блок с именем name
3. npm run guide {name} - создать блок для guide с именем name
4. npm run sprite - создать спрайт из svg картинок
5. npm run zip - создать архив с результатом сборки
6. npm run ftp - отправить файлы на ftp сервер
